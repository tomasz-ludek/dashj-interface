# dashj-interface

Add dashj-interface to your project
----------------------------

Via Gradle:
```gradle
implementation 'org.dash.dashjinterface:dashjinterface:0.0.9'
```

Via Maven:
```xml
<dependency>
  <groupId>org.dash.dashjinterface</groupId>
  <artifactId>dashjinterface</artifactId>
  <version>0.0.9</version>
</dependency>
```
