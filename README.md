# dashj-interface

Add dashj-interface to your project
----------------------------

Via Gradle:
```gradle
implementation 'org.dashj:dashj-interface:0.9.2'
```

Via Maven:
```xml
<dependency>
  <groupId>org.dashj</groupId>
  <artifactId>dashj-interface</artifactId>
  <version>0.9.2</version>
</dependency>
```
